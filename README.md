# Data Science Internship – AnalystData Lab

## Overview

This repository documents my learning journey during my **Data Science Internship at AnalystData Lab**. The internship is organized into weekly modules, each focusing on a different stage of the data science workflow.

Throughout the program, I work on multiple datasets to develop practical skills in data analysis, statistics, machine learning, and natural language processing (NLP). Each project is documented in its own dedicated repository with a detailed README.

---

## Objectives

The internship aims to strengthen practical knowledge in:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis
- Machine Learning fundamentals
- Natural Language Processing (NLP)
- Model evaluation and interpretation
- Data storytelling and reporting

---

# Weekly Progress

## Week 1 & 2 – Data Preparation, Exploration & Machine Learning

### Dataset 1 – Titanic Survival Analysis

**Topics covered**

- Data understanding
- Data cleaning
- Missing value treatment
- Exploratory Data Analysis (EDA)
- Statistical summaries
- Identification of survival factors
- Data visualization

**Outcome**

Prepared a clean dataset and explored the relationship between passenger characteristics and survival outcomes.

---

### Dataset 2 – Movie Reviews Sentiment Analysis

**Topics covered**

- Text data preprocessing
- Natural Language Processing (NLP)
- Text cleaning and normalization
- TF-IDF feature extraction
- Machine Learning classification
- Model training and evaluation

**Outcome**

Built a sentiment classification pipeline capable of predicting whether a movie review is positive or negative.

---

## Week 3 – Statistical Analysis
